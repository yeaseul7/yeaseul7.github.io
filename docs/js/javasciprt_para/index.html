Javascript의 멀티 패러다임

Javascript는 명령형, 함수형, 객체지향형, 프로토타입 기반을 지원하는 멀티 패러다임 언어이다.

명령형 프로그래밍이란?

Section titled “명령형 프로그래밍이란?”

컴퓨터가 어떻게 작업을 할지 하나하나 지시하는 방식이다.

객체지향형 (프로토타입 기반) 프로그래밍이란?

Section titled “객체지향형 (프로토타입 기반) 프로그래밍이란?”

데이터를 객체(Object) 단위로 묶어서 관리하는 방식. javascript에서는 중요한 키워드인 프로토타입 체인이 존재하는데 이것을 이해하면 좀 더 수월합니다.

함수형 프로그래밍이란?

Section titled “함수형 프로그래밍이란?”

단순하게 정의하자면 프로그래밍 패러다임 중 하나다.

자바스크립트에서는 **불변성(Immutability)**을 지향하며 외부 상태 변경을 최소화하는 것을 목표로 한다.

불변성을 지키고 상태 변경을 억제한다는 것은 코드의 예측 가능성을 높여 오류를 줄이는 것이다. 복잡한 로직 내의 조건문과 반복문을 함수형 메서드(map, filter 등)로 대체하여 가독성을 높일 수 있다.
