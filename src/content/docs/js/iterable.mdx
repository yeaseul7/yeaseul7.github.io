Symbol.iterator를 가지고 있는다. 

#### iterator protocol

next 메서드를 가지고 있으며
```javascript
{
  value:,
  done: true/false
}
```
이런식으로 뱉는다는 규칙이다.

```javascript
// 이터러블 프로토콜을 준수하는 객체
const myIterable = {
    [Symbol.iterator]() {
        let i = 0;
        
        // 이터레이터 프로토콜을 준수하는 객체
        return {
            next() {
                i++;
                if (i <= 3) {
                    return { value: i, done: false }; 
                }
                return { value: undefined, done: true }; 
            }
        };
    }
};
```
