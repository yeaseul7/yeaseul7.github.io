### Cumulative Layout Shift

사용자가 버튼을 누르려고 하는데 갑자기 이미지가 로딩되면서 버튼이 아래로 밀려나서 엉뚱한 걸 누르게 되는 경험    
CLS(Cumulative Layout Shift)

새로고침을 하다보니 데이터가 생기기전에 로딩이 돌다가 데이터가 생기면서 사진이 추가되고     
화면이 툭 하고 내려가는 현상이 발생하는 것 같다. 
따라서 skeleton 방식으로 이를 해결하고자 하였다.

### next/dynamic

React.lazy()와 Suspense를 합쳐놓은 Next.js의 기능이다.

```javascript
import dynamic from 'next/dynamic'
 
// '../components/header'를 나중에(Lazy) 불러옵니다.
// 불러오는 동안에는 'Loading...' 문구를 보여줍니다.
const DynamicHeader = dynamic(() => import('../components/header'), {
  loading: () => <p>Loading...</p>,
})
 
export default function Home() {
  return <DynamicHeader />
}
```

위의 방식으로 lottie를 보여주는 부분에 dynamics를 적용하였다.
