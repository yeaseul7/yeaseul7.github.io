### github 프로필 방문자 수 추가하기

깃허브 프로필은 내 github repository에 있는 readme.md 파일을 수정하면 된다.

#### 방문자 수를 표기하는 방법

사이트 : https://hits.seeyoufarm.com/

위 사이트에 들어가서 자신의 repository 경로를 입력해준다.
