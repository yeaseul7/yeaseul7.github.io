### INTRODUCE

웹/앱 서비스 개발을 중심으로 프론트엔드 개발을 담당해 온 이예슬입니다. React, TypeScript, Next.js를 주력으로 사용하며, 사용자 흐름에 맞는 화면을 구현하고 유지보수하기 좋은 컴포넌트 구조를 고민합니다.

CRM 개발에서는 기존 Vue + CDN 기반 화면 구조를 React 앱 기반 구조로 전환하고, 분산되어 있던 화면 개발 방식을 컴포넌트 중심으로 정리했습니다. 또한 Azure CDN 마이그레이션과 불필요한 호출 로직 정리를 통해 렌더링 속도를 10초에서 3초로 개선하며 사용자 체감 성능을 높였습니다.

저는 기능을 단순히 구현하는 것보다, 서비스가 해결하려는 문제와 사용자의 실제 흐름을 이해하는 것을 중요하게 생각합니다. 비즈니스 요구사항을 기술적으로 풀어내기 위해 팀과 적극적으로 소통하며, 안정적이고 사용하기 쉬운 서비스를 만드는 프론트엔드 개발자로 성장하고자 합니다.
